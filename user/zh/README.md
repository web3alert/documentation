# 欢迎使用 Web3alert

Web3alert 是一个用于创建和接收 Web3 通知的服务。

它可以把 blockchain/runtime 事件、外部数据和用户定义的条件转换成清晰的 alerts，并发送到 Telegram、Discord、Slack、webhooks 和其他投递渠道。

如果你想使用已有 alerts，请从 [Subscriptions](subscriptions.md) 开始；如果你想创建自己的 integration，请从 [Projects](projects.md)、[Triggers](triggers.md) 和 [Templates](templates.md) 开始。
